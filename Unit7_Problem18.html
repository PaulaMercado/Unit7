When this code is run it returns an alert message "HEY YOU!"
and an error message for unexpected token. The reason for this
is because the statement shout(); runs the function by the name
shout. The line after that alerts message which is a local variable
inside the function. Unless there's another variable called message,
that is global and doesn't belong to a function it will crash and
return a message in the runner that says:
"Uncaught ReferenceError: message is not defined"
